Scalable Swarm Robotics with Vectors in GPS-Denied Environments

Mihir Parekh & Kshitij Jerath

Emergent Dynamics, Control, and Analytics Lab @ University of Massachusetts Lowell

Problem

Biologically inspired swarming algorithms have seen significant development in simulation environments; however real-world experimental validation is essential for verifying their robustness, scalability, and performance under uncertainty. Simulation often abstracts away critical real-world constraints such as sensor noise, communication latency, hardware limitations, and unstructured environments. Experimental testbeds help bridge this gap, providing vital feedback on swarm theory and design.

Task

To increase accessibility to experimental swarm testbeds, I proposed building a swarm using commercially available Anki Vector robots. The Anki Vector is a compact differential-drive robot equipped with a camera, IMU, microphone, and accelerometer. The core research question is:

Can we build a real-world swarm system using commercially available robots in GPS-denied environments, relying purely on perception?

Approach

We developed a centralized perception system to support the design of decentralized swarm behaviors.

Modules Implemented:

  • Online Vector Detection
  • Inter-Robot Distance Estimation
  • Target Following
  • Fixed Landmark Localization
  • Real-Time Pose Estimation using Sensor Fusion
  • Current Work: Multi-robot localization, scaling up system robustness and communication for larger swarms.
